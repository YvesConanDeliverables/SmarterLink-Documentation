Class ProtocolPayloadNameFormatter

Namespace
SmarterLink.Core.Formatting
Assembly
SmarterLink.Core.Formatting.dll

Provides stable wire name and documentation name resolution for SmarterLink protocol payload types. All name forms are derived from PayloadTypeNameAttribute when present on the type, falling back to the C# type name when the attribute is absent.

GetSchemaLongName(Type)

Returns the long schema name for the given payload type, used in protocol schema and enum exports. Returns the Name if the attribute is present; otherwise camel-cases each segment of the nested type path and joins them with / (e.g. JobEvents.ActiveJobUpdated produces jobEvents/activeJobUpdated).
